# fastreg
DERO ultra fast new account generation and registration

## Requires go v1.17

## To build / run:

### 1) Clone repository:
	$ git clone https://github.com/deroholic/fastreg.git

### 2) Enter directory:
	$ cd fastreg

### 3) Build and run:
	$ go get
	$ go build
	$ ./fastreg
